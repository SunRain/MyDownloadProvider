MyDownloadProvider
========
android library+gradle+混淆<br/>
基于android4.4版本的DownloadProvider构建，可作为library直接供其他项目使用。

library说明:
-----------------------------------
WdUtil：封装了util类，作为所有project的基础library;<br/>
DownloadProvider-lib：基于android源码的下载模块;<br/>
